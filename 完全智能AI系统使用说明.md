## 🔧 快速开始

### 1. 准备文件
```bash

├── 电影1.srt
├── 电影2.txt
└── 电影3.srt

# 将对应视频文件放入
movie_videos/
├── 电影1.mp4
├── 电影2.mkv
└── 电影3.avi
```


  - 通义千问
  - 智谱AI

  - Claude

### 4. 自动处理

- 创建详细分析报告
